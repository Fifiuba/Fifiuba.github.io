Guía de cómo utilizar el Back Office

Para poder ingresar al Back Office primero se debe logear con la cuenta super admin

coreoElectronico: adminfiuba@fiuba.com contra: adminfiuba

El login pose la opcion de mantener la session abierta a pesar de cerrar el navegador. En caso de no marcar la opcion, esta se mantendra mientras que no se cierre la pestaña

Home Page

En el home podemos encontrar todas las tarjetas correspondientes a las distintas funcionalidades

Gestion Usuarios

En esta seccion se listarán todos los pasajeros del sistema mostrando sus nombres, dirreciones de correo electronico, telefonos y edades. Estos datos estarán dispuestos en una tabla paginada en la cual el administrador puede navegar con el paginador. En esta seccion se puede ver en detalle los datos de cada pasajero con el icono de informacion y tambien se podra bloquear o desbloquear al usuario con el boton de bloqueo.

Ver usuario

Al ingresar a informacion mas detallada de un usuario el administrador podra ver en detalle la siguiente informacion del pasajero

  1. Datos Generales
    1. Id
    2. Nombre
    3. Edad
  2. Direccion
  3. Contacto
    1. Correo Electronico
    2. Telefono
  4. Callificacion

Gestionar Choferes

En esta seccion se listarán todos los choferes del sistema mostrando sus nombres, dirreciones de correo electronico, telefonos y edades. Estos datos estarán dispuestos en una tabla paginada en la cual el administrador puede navegar con el paginador. En esta seccion se puede ver en detalle los datos de cada chofer con el icono de informacion y tambien se podra bloquear o desbloquear al chofer con el boton de bloqueo.

Ver chofer

  3. Contacto

    2. Telefono
  4. Callificacion

Gestionar admins

En esta seccion se listarán todos los administradores del sistema mostrando sus nombres y dirreciones de correo electronico Estos datos estarán dispuestos en una tabla paginada en la cual el administrador puede navegar con el paginador.

Registrar Admin

En esta seccion se podra registrar a un administrador que para poder crearse una cuenta debera cargar los siguientes datos:

  1. Nombre
  2. Apellido
  3. Correo Electronico: Es el identificador unico del usuario en la aplicacion y no podra ser modificado. Ingresar un email válido.
  4. Contraseña: La contraseña debe tener un minimo de

Perfil

En esta seccion podemos visualizar nustros datos personales: Nombre, Apellido y Correo Electronico. En caso de queder modifcar algun dato se debe apretar el boton editar perfil

Editar Perfil

En esta seccion, podemos modificar los datos personales del administrador: nombre y apellido. El correo electronico es el unico dato que no se puede modificar ya que es el identificador unico que posee el administrador.

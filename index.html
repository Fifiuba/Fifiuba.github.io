Guía de cómo utilizar la Aplicación

Introducción

Dentro de la aplicación el usuario podrá desempeñarse como pasajero y como chofer, con el mismo correo electronico pero debera loggearse con el rol que desee ingresar. Estos roles compartiran los datos generales del usuario, pero tendran datos especificos segun el tipo del rol con el que ingresen.

Además, en la aplicacion el pasajero podra buscar distintos viajes a realizar y solicitarlos mientras que el chofer recibira solicitudes de viajes cercanos a el, y podra aceptarlos o rechazarlos. Una vez comenzado el viaje y finalizado se cobrara el viaje y se podra calificar tanto al chofer como al pasajero.

En la aplicación, también habrá un sistema de usuarios que podrá verse explicado en las siguientes secciones para los registros y logins de usuarios tanto con email y contraseña, como con un login federado con una cuenta de Google.

Registro

Para registrar a un usuario desde la pantalla inicial de la aplicacion se debe apretarl el boton de "No tenes cuenta? Registrate" para poder ingresar la informacion de registro. Una vez que se entro a la seccion de registrar usuario, el usuario podra crearse una cuenta con email y contraseña. Debera ingresar los siguientes campos:

  • Nombre
  • Correo Electronico
  • Telefono
  • Edad
  • Contraseña
Ademas debera indicar como desea registrarse: pasajero o chofer utilizadon el boton swipe.
Una vez que ingresa los datos correspondientes, deberá pulsarse el botón de Continuar para finalizar el registro. En caso de errores se mostrará descriptivamente el fallo por el cuál no puede realizarse el registro. En caso de que si posea una cuenta y desee ingresar a la aplicacion con ella debe apretar el boton de "iniciar sesion"

Login

Para poder ingresar a la aplicacion, el usuario deberá ingresar sus credenciales, el email , la conseña y con que rol desea ingresar. En caso de que estas se validad y sean correctas, el usuario podra ingresar a la aplicación generandose una session redirigiendo al usuario a la página de inicio.

El usario tambien podra iniciar session con google aprendado el boton de "GOOGLE"

Aclaración: En caso de que el usuario se encuentre bloqueado no podrá realizar un login en la aplicación y deberá comunicarse con un administrador.

Recupero De contraseña

En caso de que el usuario no recurde su contraseña, este debera apretar el boton "¿Has olvidado tu contraseña?" donde se le pedira que ingrese su email. El usuario deber apretra el boton de enviar link para que se le pueda mandar el link de recupero de contraseña al emial ingresado.

Home Pasajero

Al ingresar a la aplicacion, si el usuario ingresa por primera vez a la aplicacion no se mostrar ningun viaje realizado. Cada vez que desee volver a home el pasajero debera apretar el boton con imagen de una casa.

Una vez que el usuario realice un viaje, el usuario va poder visualizar sus ultimos viajes realizados aprendado el boton buscar ultimos viajes

Perfil Pasajero

Si el pasajeo desea visualizar su perfil, este debe apretar el boton "perfil" que se encuentra en la session inferior a la derecha. En esta seccion el pasajero puede visualuzar los siguientes datos:

  • Nombre
  • Correo Electronico
  • Telefono
  • Edad
  • Direccion default

Editar Perfil Pasajero

Desde la seccion "Perfil" el pasajero podra realizar una edición de perfil, aprentando el boton editar. En esta seccion se le permite al pasajero modificar su nombre, edad, telefono, direccion y avatar. Para que los datos se modifiquen efectivamente, el pasajero debera apretar el boton guardar

Viaje Pasajero

Si el pasajero desea realizar un viaje, este debera apretar el boton "viajes" donde sera redirigio a una pantalla donde el usuario debera ingresar una direccion de origen y una de destino.

Una vez ingresado el origen y destino del viaje, para poder visualizar el precio del viaje el pasajero debra apretar el boton de buscar viaje.

Si el pasajero desea realizar el viaje, debera apretar el boton de solicitar viaje donde se registrara su solcitud de viaje si se comenzara a buscar a un chofer.

Espera chofer

Una vez que el pasajero solicita el viaje, se lo rederigida a una pantalla de espera donde esperara a que halla un chofer disponible. El pasajero podra ver los siguientes datos:

  • Origen viaje
  • Dstino viaje
  • Precio
En caso de que desee cancelar el viaje, el pasajero debera apretar el boton de cancelar viaje.

Home chofer

Al ingresar a la aplicacion, el chofer se encontra en la pantalla de inicio donde podra buscar viajes a realizar cercanos a el aprentando el boton buscar viajes. Cada vez que desee volver a home el chofer debera apretar el boton con imagen de una casa.

Una vez que el chofer busque nuevos viajes, se le mostrara la descripciones de los viajes mas cercanos. El chofer podra aceptar los viajes o rechazarlos.

Perfil chofer

Si el chofer desea visualizar su perfil, este debe apretar el boton "perfil" que se encuentra en la session inferior a la derecha. En esta seccion el chofer puede visualuzar los siguientes datos:

  • Nombre
  • Correo Electronico
  • Telefono
  • Edad
  • Modelo del Vehiculo
  • Patente del Vehiculo

Editar Perfil Chofer

Desde la seccion "Perfil" el chofer podra realizar una edición de perfil, aprentando el boton editar. En esta seccion se le permite al pasajero modificar su nombre, edad, telefono, modelo, patente y avatar. Para que los datos se modifiquen efectivamente, el chofer debera apretar el boton guardar

Viaje chofer

Si cuando el chofer busca nuevos viajes solicitados y acepta un viaje se le mostrara el mapa desde donde el se encuentra. Para poder ir al lugar de origen del viaje y que se le muestre el camino debera apretar el botn ir.

Una vez que el chofer llega al lugar de origen debera apretar el boton comenzar. Al chofer se le mostrara el camino del viaje hasta llegar a destino.

Una vez que el chofer llega a destino este debe apretar el boton finalizar para efectivamente terminar el viaje y poder cobrarlo.

Viaje Pasajero

Cuando un chofer le acepta el viaje al pasajero y llega al lugar de origen del viaje. Se le mostrara al pasajero el camino que estar realizando su chofer hasta llegar a destino, indicando el tiempo transcurrido.

Pago pasajero

Una vez que finaliza el viaje, se le informa al pasajero que se ha realizado el deposito del viaje al chofer y que podra calificarlo si es que lo desea o directamente redirigirse a inicio.

Pago chofer

Una vez que el chofer finalice el viaje, se le informa al chofer que ha recibido el pago del viaje y que podra calificar al pasajero si es que lo desea o directamente redirigirse a inicio.

Calificacion

Una vez hecho el pago/deposito tanto el chofer como el usuario podran calificar al otro y dejar un comentario. Para eso deben apretar el boton de calificar y selecionar cuantas estrellas desean otorgarle al otro y escribir un comentario si lo desean. Una vez hecho esto apredan aceptar.

Wallet

Si el pasajero o el chofer desean ver su wallet, deberan apretar el boton billetera. En esta seccion podran ver su nombre y su saldo dispóbible.
